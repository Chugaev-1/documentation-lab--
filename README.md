# Програма для відстеження особистих цілей

## Опис проєкту
Цей проєкт — проста програма для відстеження особистих цілей користувача.
Вона допомагає людям ставити цілі, контролювати прогрес та підтримувати мотивацію.

Документація створена в межах лабораторної роботи з теми «Документація в ІТ».

## Основні функції
- Створення особистих цілей
- Перегляд списку цілей
- Відмітка виконаних цілей
- Перегляд прогресу

## Для кого цей продукт
- Студенти
- Люди, які хочуть покращити самодисципліну
- Користувачі, що планують свій розвиток
